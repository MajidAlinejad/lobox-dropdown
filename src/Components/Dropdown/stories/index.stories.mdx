# Dropdown

The Main Organization for Drop Down Component is `Dropdown`, we have two Type of child node :<br/>

#### `Selector`

The select trigger component for dropdown.

#### `OptionContainer`<br/>

The Container for options.It handle position and z-index.

#### `OptionItem`<br/>

The options that take two parameter : `name` and `value`
